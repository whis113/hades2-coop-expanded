* This is test version of Hades 2 coop mod, which is in early development stage. It may contain bugs and may cause crashes.
* Make sure to backup your save files before installing the mod. The save files are located in "C:\Users\<USER>\Saved Games\Hades II".

To install the mod, follow these steps:
* Run the "install.ps1" file.
* Select the path to Hades2.exe
